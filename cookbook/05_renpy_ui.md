# 05. Ren'Py UI와 화면 구성

[목차로 돌아가기](README.md)

이 장은 **메인 메뉴, 대화창, 포커 테이블, 결과 화면**을 어떻게 읽고 수정해야 하는지 정리한다.

UI는 숫자 몇 개만 만져도 쉽게 무너진다.  
그래서 이 장은 “어디를 손대면 무엇이 바뀌는지”를 계층별로 설명한다.

## 1. UI 계층 구조

### `screens.rpy`

역할:

- 메인 메뉴
- 일반 say screen
- 기본 네비게이션

여기서 책임지는 것:

- 메인 메뉴 패널 폭
- 메뉴 버튼 글자 크기
- 타이틀 네온
- 일반 대화창의 기본 틀

### `gui.rpy`

역할:

- 공통 스케일
- 텍스트 크기
- 대화창 치수
- 이름표 치수

여기서 책임지는 것:

- 전체 UI의 기본 크기감
- 대화창 높이
- 이름표가 본문을 덮지 않게 하는 기준
- `gui_scale()`이 실제 좌표를 한 번 더 줄이지 않도록 하는 기준

### `poker_ui.rpy`

역할:

- 포커 전용 화면
- 카드 렌더링
- HUD
- 액션 버튼
- 유틸 버튼
- 결과 화면

여기서 책임지는 것:

- 카드가 충분히 큰지
- 행동 버튼과 유틸 버튼이 분리되어 있는지
- 결과 화면 카드가 안 잘리는지

## 2. 메인 메뉴

파일:

- `llmoker/game/screens.rpy`

메인 메뉴에서 먼저 보는 것:

- 타이틀이 실제로 읽히는가
- 네온 글로우가 제목보다 먼저 튀지 않는가
- 왼쪽 검은 패널이 버튼 글자보다 충분히 넓은가
- 메뉴 글자가 너무 작지 않은가
- 배경 캐릭터가 탁해지지 않았는가
- 도움말/정보/환경 설정 같은 서브 메뉴가 왼쪽 레일과 겹치지 않고 중앙 패널로 뜨는가

자주 생기는 문제:

- 타이틀 글로우를 과하게 올려 읽기 어려움
- 패널 폭 축소로 글자 눌림
- 오버레이가 너무 진해 캐릭터 흐려짐
- 이미 1024 기준으로 잡힌 값을 `gui_scale()`로 다시 줄여 버튼과 타이틀이 필요 이상 작아짐
- 정보와 환경 설정에 Ren'Py 엔진 설명이 섞이면 처음 보는 사용자가 게임 도움말로 읽기 어렵다.
- 도움말, 정보, 환경 설정은 게임 조작과 플레이 규칙 위주로 유지하고, 엔진 설명은 문서로 분리한다.

## 3. 일반 대화창

주요 파일:

- `llmoker/game/gui.rpy`
- `llmoker/game/screens.rpy`

중요 요소:

- 검은 대화 패널 높이
- 이름표 폭
- 이름표 y 위치
- 본문 시작 위치
- 본문 폰트 크기

감사 기준:

- 이름표가 본문 첫 줄을 덮지 않는가
- 검은 패널이 너무 작아 대사가 눌리지 않는가
- 반대로 너무 커서 화면을 과하게 가리지 않는가
- `gui.init(1024, 576)` 이후에도 다시 축소되고 있지 않은가

## 4. 포커 플레이 화면

주요 파일:

- `llmoker/game/poker_ui.rpy`

### 카드 영역

중요 요소:

- 카드 zoom
- 카드 아래 라벨 글자 크기
- 카드 간 간격

감사 기준:

- 숫자와 무늬가 실제로 읽히는가
- 카드가 너무 작아지지 않았는가

### 상단 HUD

들어가는 정보:

- 현재 페이즈
- 팟
- 양측 칩
- 상대 AI 모드
- 현재 족보
- 상태 텍스트

감사 기준:

- 상단 정보 바가 카드 패널을 누르지 않는가
- 페이즈 글자가 깨지기 직전으로 작지 않은가
- 상태 문구가 너무 많이 눌리지 않는가

### 하단 버튼

원칙:

- 게임 행동 버튼은 좌하단
- 유틸 버튼은 우하단

게임 행동 버튼:

- 체크
- 콜
- 베팅
- 레이즈
- 폴드

유틸 버튼:

- 로그 보기
- 저장
- 불러오기
- 환경 설정
- 메인 메뉴

이 둘이 한 줄에 섞이면 플레이 감각이 무너진다.

## 5. 결과 화면

주요 파일:

- `llmoker/game/poker_ui.rpy`

결과 화면이 반드시 보여줘야 하는 것:

- 승패 제목
- 결과 설명
- 사야의 패
- 당신의 패
- 현재 족보
- 현재 스택

배치 원칙:

- 상단: 결과 제목과 설명
- 중단: 상대 패 상단 / 플레이어 패 하단
- 좌하단: `다음 라운드`, `메인 메뉴`
- 우하단: 로그/저장/불러오기/환경 설정 같은 유틸

자주 생기는 문제:

- 결과 패널이 너무 낮아 카드가 잘림
- 카드 zoom이 줄어 숫자가 안 보임
- 텍스트는 많은데 패널 높이가 부족함
- 로그 화면에서 게임으로 돌아오는 버튼이나 닫기 경로가 없으면 UI가 사실상 막힌다.

## 6. 어떤 파일을 먼저 열어야 하나

### 메인 메뉴가 안 읽힌다

먼저:

- `screens.rpy`

그 다음:

- `gui.rpy`

### 대화창이 이상하다

먼저:

- `gui.rpy`

그 다음:

- `screens.rpy`

### 카드와 HUD가 너무 작다

먼저:

- `poker_ui.rpy`

### 결과 화면이 잘린다

먼저:

- `poker_ui.rpy`

그 다음:

- `gui.rpy`
